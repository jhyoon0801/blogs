Spring MVC 기초 - 1

Spring MVC 기초 - 1 Spring MVC에 관련한 기초 내용을 정리해 나가자. web application 개발 관련 기본 정리 web server란? HTTP 프로토콜을 사용하는 클라이언트로부터의 요청을 받아서, HTML(그림, CSS, 자바스트립트 등의 리소스를 포함한) 문서와 같은 리소스를 반환하는 컴퓨터 프로그램이다. web application이란? 정적 데이터(정적 HTML 문서 등)가 아니라, 각각의 요청에 따라 동적 데이터(동적 HTML문서 등)를 생성하는 프로그램이며, 이 프로그램의 동작은 web server의 호출에 의해 수행된다. web application은 web server와는 별개의 프로세스로도 동작할 수 있고, 동일한 프로세스로도 동작할 수 있다.
